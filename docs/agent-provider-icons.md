# Agent Provider Icons

Agent status provider icons include compact marks from these upstream sources:

- OpenCode: `opencode-logo-dark-square.svg` from `anomalyco/opencode`, MIT license.
  Source: https://github.com/anomalyco/opencode/blob/dev/packages/console/app/src/asset/brand/opencode-logo-dark-square.svg
- Pi: badge SVG from the official Pi press kit, MIT license.
  Source: https://pi.dev/press-kit
- Hermes Agent: ACP registry icon from `NousResearch/hermes-agent`, MIT license.
  Source: https://github.com/NousResearch/hermes-agent/blob/main/acp_registry/icon.svg

These marks are used only to identify the installed CLI agent provider in Terax UI.
